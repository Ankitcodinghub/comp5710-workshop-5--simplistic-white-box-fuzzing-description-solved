Develop a simple fuzzer to test an existing implementation of a simple calculator

Targeted Courses

Software Quality Assurance

Activities

Pre-lab Content Dissemination

White-box fuzzing is a fuzzing technique where the source code of a software is leveraged to generate test cases. The goal of these test cases is to find bugs in the source code, so that these bugs are found and fixed early at the development stage. As part of this workshop we will develop a simple fuzzer that fuzzes an existing implementation of a simple calculator.
